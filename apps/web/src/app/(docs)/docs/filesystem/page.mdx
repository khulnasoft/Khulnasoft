# Filesystem

Each KhulnaSoft Sandbox has its own isolated filesystem. The [Hobby tier](https://khulnasoft.com/pricing) sandboxes come with 1 GB of the free disk space and [Pro tier](https://khulnasoft.com/pricing) sandboxes come with 5 GB.

With KhulnaSoft SDK you can:
- [Read and write files to the sandbox.](/docs/filesystem/read-write)
- [Watch directory for changes.](/docs/filesystem/watch)
- [Upload data to the sandbox.](/docs/filesystem/upload)
- [Download data from the sandbox.](/docs/filesystem/download)
