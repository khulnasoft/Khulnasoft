## khulnasoft sandbox


work with sandboxes

### Usage

```bash
khulnasoft sandbox [options] [command]
```
## khulnasoft sandbox connect


connect terminal to already running sandbox

### Usage

```bash
khulnasoft sandbox connect [options] <sandboxID>
```


## khulnasoft sandbox list


list all running sandboxes

### Usage

```bash
khulnasoft sandbox list [options]
```


## khulnasoft sandbox kill


kill sandbox

### Usage

```bash
khulnasoft sandbox kill [options] [sandboxID]
```

### Options


 - `-a, --all: kill all running sandboxes `


## khulnasoft sandbox spawn


spawn sandbox and connect terminal to it

### Usage

```bash
khulnasoft sandbox spawn [options] [template]
```

### Options


 - `-p, --path <path>: change root directory where command is executed to <path> directory `
 - `--config <khulnasoft-toml>: specify path to the KhulnaSoft config toml. By default KhulnaSoft tries to find ./khulnasoft.toml in root directory. `


## khulnasoft sandbox logs


show logs for sandbox

### Usage

```bash
khulnasoft sandbox logs [options] <sandboxID>
```

### Options


 - `--level <level>: filter logs by level (DEBUG, INFO, WARN, ERROR). The logs with the higher levels will be also shown. [default: INFO]`
 - `-f, --follow: keep streaming logs until the sandbox is closed `
 - `--format <format>: specify format for printing logs (json, pretty) [default: pretty]`
 - `--loggers [loggers]: filter logs by loggers. Specify multiple loggers by separating them with a comma. `
