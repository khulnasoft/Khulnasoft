## khulnasoft auth


authentication commands

### Usage

```bash
khulnasoft auth [options] [command]
```
## khulnasoft auth login


log in to CLI

### Usage

```bash
khulnasoft auth login [options]
```


## khulnasoft auth logout


log out of CLI

### Usage

```bash
khulnasoft auth logout [options]
```


## khulnasoft auth info


get information about the current user

### Usage

```bash
khulnasoft auth info [options]
```


## khulnasoft auth configure


configure user

### Usage

```bash
khulnasoft auth configure [options]
```
